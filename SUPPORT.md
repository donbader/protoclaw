# Support

## Getting Help

- **Bug reports**: [Open an issue](https://github.com/donbader/anyclaw/issues/new?template=bug_report.yml)
- **Feature requests**: [Open an issue](https://github.com/donbader/anyclaw/issues/new?template=feature_request.yml)
- **Questions & discussion**: [Open an issue](https://github.com/donbader/anyclaw/issues) with your question

## Before Filing an Issue

1. Search [existing issues](https://github.com/donbader/anyclaw/issues) to avoid duplicates
2. Include your anyclaw version (`anyclaw --version` or check `Cargo.toml`)
3. For bugs, use the bug report template — it asks for reproduction steps and logs

## Security Vulnerabilities

Do **not** file a public issue. See [SECURITY.md](SECURITY.md) for responsible disclosure instructions.
