# Code of Conduct

## Our Pledge

We are committed to providing a friendly, safe, and welcoming environment for all contributors, regardless of experience level, gender identity, sexual orientation, disability, personal appearance, body size, race, ethnicity, age, religion, or nationality.

This project adopts the [Rust Code of Conduct](https://www.rust-lang.org/policies/code-of-conduct). All participants in anyclaw project spaces are expected to follow it.

## Scope

This Code of Conduct applies to all project spaces, including:

- GitHub issues, pull requests, and discussions
- Code review comments
- Any other communication channels associated with anyclaw

## Enforcement

Instances of unacceptable behavior may be reported by [opening a private security advisory](https://github.com/donbader/anyclaw/security/advisories/new) on this repository, or by contacting a project maintainer directly via GitHub. All reports will be reviewed and handled confidentially.

Project maintainers who do not follow or enforce the Code of Conduct may face temporary or permanent consequences as determined by other members of the project's leadership.
