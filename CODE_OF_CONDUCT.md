# Code of Conduct

## Our Pledge

This project follows the [Rust Code of Conduct](https://www.rust-lang.org/policies/code-of-conduct).

The Rust CoC applies to all project spaces, including GitHub issues, pull requests, discussions, and any other communication channels associated with protoclaw.

## Enforcement

Instances of unacceptable behavior may be reported to the project team at [conduct@protoclaw.dev](mailto:conduct@protoclaw.dev). All reports will be handled confidentially.
