  <action>
    Add round-trip serialization tests for EVERY public struct and enum in anyclaw-sdk-types (per D-07).

    Pattern for each test (use rstest with BDD naming):
    ```rust
    #[rstest]
    #[test]
    fn when_{type_name}_serialized_and_deserialized_then_round_trips_correctly() {
        let original = TypeName { field1: value1, field2: value2, ... };
        let json = serde_json::to_value(&original).unwrap();
        let restored: TypeName = serde_json::from_value(json).unwrap();
        assert_eq!(original, restored);
    }
    ```

    For enums, test each variant separately. For types with optional fields, test both present and absent cases.

    Types to cover in acp.rs: ClientCapabilities, InitializeParams, InitializeResult, McpServerInfo, SessionNewParams, SessionNewResult, ContentPart (all variants), SessionPromptParams, SessionCancelParams, SessionForkParams, SessionForkResult, SessionListParams, SessionListResult, SessionInfo, SessionLoadParams, ToolCallStatus (all variants), SessionUpdateType (all variants), SessionUpdateEvent.

    Types to cover in channel.rs: ChannelCapabilities, ChannelInitializeParams, ChannelInitializeResult, DeliverMessage, PeerInfo, ChannelSendMessage, ThoughtContent, ContentKind (all variants), AckNotification, AckLifecycleNotification, ChannelAckConfig, ChannelRespondPermission, SessionCreated.

    Types to cover in permission.rs: PermissionOption, PermissionRequest, PermissionResponse, ChannelRequestPermission.

    Types to cover in channel_event.rs: ChannelEvent (all variants).

    Types to cover in session_key.rs: SessionKey (Display + FromStr round-trip).

    Add tests to existing `#[cfg(test)] mod tests` blocks in each file. Some files already have partial tests — add the missing round-trip tests alongside them.

    NOTE: Some types may not derive PartialEq. If so, add `#[derive(PartialEq)]` to the type (it's a test-only need but PartialEq is reasonable for wire types). Alternatively, compare the serialized JSON: `assert_eq!(serde_json::to_value(&original), serde_json::to_value(&restored))`.
