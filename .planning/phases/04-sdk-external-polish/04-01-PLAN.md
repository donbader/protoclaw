## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| SDK trait → implementor | External code implements SDK traits — typed signatures enforce correct data shapes at compile time |
| JSON-RPC stdio → harness | Raw JSON from subprocess stdin parsed into typed structs at harness boundary |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-04-01 | T (Tampering) | AgentAdapter hooks | mitigate | Typed params prevent malformed JSON from reaching adapter — deserialization fails at boundary before hook is called |
| T-04-02 | I (Info Disclosure) | Tool execute I/O | accept | Tool I/O stays as Value (D-03) — tool authors control their own schema; no PII risk from typing |
| T-04-03 | T (Tampering) | ChannelHarness JSON parse | mitigate | Parse stdin as JsonRpcMessage (typed) instead of raw Value — malformed messages rejected at parse boundary |
</threat_model>
